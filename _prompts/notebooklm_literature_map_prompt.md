# NotebookLM 学术争议地图 Prompt

## 目标

只根据我上传的 CSSCI 论文，整理惩罚性赔偿司法适用的学术争议地图。

## 严格要求

1. **只根据上传的论文内容**，不补充外部信息。
2. **不编造论文名称、作者、观点。**
3. 如果某篇论文没有涉及某个争议问题，不要强行归入。
4. 输出必须包含来源标注（哪篇论文、哪个观点）。

## 输出格式

请按以下表格格式输出：

| 争议问题 | 代表观点 A | 代表观点 B | 主要理由 | 对判例梳理的启发 | 可转化为案例卡字段 | 是否适合放入 pre |
|---|---|---|---|---|---|---|

## 常见争议问题参考

以下问题仅供参考，请根据论文实际内容调整：

1. 惩罚性赔偿的正当性基础是什么？
2. "故意"和"明知"如何区分？
3. "情节严重"的判断标准是什么？
4. 赔偿基数如何确定？
5. 倍数裁量应考虑哪些因素？
6. 惩罚性赔偿与精神损害赔偿能否并用？
7. 不同领域（知产、产品责任、环境、消费者）的惩罚性赔偿有何异同？
8. 证明责任如何分配？
9. 惩罚性赔偿的适用限制是什么？

## 使用方式

请上传论文 PDF 或 extracted_md 文件，然后输入：
```
请根据上传的论文生成学术争议地图。
```
