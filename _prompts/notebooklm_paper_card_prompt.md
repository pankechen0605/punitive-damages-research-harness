# NotebookLM 论文卡生成 Prompt

## 目标

根据单篇论文生成论文卡初稿。

## 严格要求

1. **不要总结全文流水账。** 重点提取对判例梳理有用的框架。
2. **明确哪些观点可以转化为案例观察字段。**
3. 论文用于观察框架，**不用于证明案件事实**。
4. 不确定的内容写"待核验"。
