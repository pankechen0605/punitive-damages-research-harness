# Public Harness Note

This repository is a sanitized workflow/documentation harness for the local `punitive-damages-research` project.

It is intended for ChatGPT/Codex/GitHub-based review of project structure, workflow, templates, prompts, and documentation.

It intentionally excludes:

- raw legal materials;
- full case texts;
- PDF papers;
- database clippings;
- classroom notes;
- generated research cards containing source-specific legal content unless manually added later.

Use the local private workspace as the source of truth for raw materials.
