本文是当前 topic instance 的约束文件，不是通用模板规则。若以后更换研究主题，应新建或更新对应 topic config，而不是直接复用本文的具体范围判断。

## 范围定位

本次不做惩罚性损害赔偿全景综述。

老师提示“惩罚性损害赔偿”主题太大，应当细化到一个具体领域或具体问题。基于当前项目材料和课堂提示，本项目当前收束到：

```text
生态环境侵权惩罚性赔偿
```

## 当前核心切口

当前主线围绕三个实务问题展开：

1. 适用门槛：故意 + 严重后果
2. 计算基数
3. 行政处罚 / 刑事责任与民事惩罚性赔偿的竞合

这些切口用于指导案例筛选、案例卡字段、横向比较表和最终展示材料。

## 不进入主线的内容

麦当劳案只作为 30 秒引入，不展开比较法。

精神损害隐性惩罚功能只作为结尾延伸，不进入主线。

知识产权、产品责任、消费者权益、食品药品安全等领域不作为本次主线展开，除非人工明确调整选题。

## 当前案例检索目标

当前案例检索目标是：

```text
6-8 个生态环境惩罚性赔偿候选案例
```

候选案例应优先服务以下观察需求：

- 是否支持惩罚性赔偿
- 主观故意如何认定
- 严重后果如何认定
- 赔偿基数如何计算
- 倍数如何裁量
- 行政处罚或刑事责任是否影响民事惩罚性赔偿
