# 横向比较表：{{topic_name}}

## 使用说明

本表的列应从观察要素模型派生，不应手工随意添加。

先在 `2_framework/adjudication_factors` 中锁定 factor model，再将 factor 派生为主题比较列。

## 列选择规则

- P0 要素派生为横向比较主列。
- P1 要素派生为辅助列。
- P2/P3 要素通常不进入主表，可进入备注或附表。
- 如果某个 P0 要素在 pilot cases 中无法稳定填写，需要回到 factor weighting model 调整。
