# 案例卡字段派生表：{{topic_name}}

Universal fields 是所有案例都需要的字段。

Topic-specific fields 由当前主题观察要素派生。

主题字段不能未经检查就复制到其他主题。

P0 字段一般为 Required。

P1 字段一般为 Strongly Recommended。

P2/P3 字段一般为 Optional。

是否必填最终由人工根据主题决定。不得因为 AI 认为某字段重要就自动设为 P0。
