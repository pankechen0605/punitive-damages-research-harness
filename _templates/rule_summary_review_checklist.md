# Rule Summary Review Checklist

- [ ] Every conclusion traces back to cards.
- [ ] No invented case / statute / paper.
- [ ] No paper view is written as a court rule.
- [ ] No ordinary case is treated as a rule anchor by itself.
- [ ] Every tentative pattern states sample scope.
- [ ] Every final legal conclusion has human sign-off.
- [ ] Contradicting materials have been considered.
- [ ] Strong language has been reviewed.
- [ ] Uncertain content is marked “待核验”.
- [ ] Output-layer language does not strengthen synthesis-layer conclusions.
